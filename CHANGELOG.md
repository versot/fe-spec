# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-01-01

### Features

- Initial release of versot-fe-spec
- versot-fe-lint: CLI tool for one-click lint setup
- versot-fe-eslint-config: ESLint configurations for JS/TS/React/Vue/Node
- versot-fe-stylelint-config: Stylelint configurations for CSS/SCSS/Less
- versot-fe-commitlint-config: Commitlint configuration for Git
- versot-fe-markdownlint-config: Markdownlint configuration for docs
- versot-fe-eslint-plugin: Custom ESLint rules
